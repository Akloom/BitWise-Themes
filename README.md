# BitWise Themes

**BitWise Themes** - это набор кастомных тем для вашего редактора.

## **Установка**

Для установки плагина можно воспользоваться менеджером расширений Visual Studio Code и найти плагин BitWise Themes

## **Темы**

**Dark Themes:**
